# アプリ名
QuizPar5 (クイズパーファイブ)

# 概要
 ゴルフのクイズが出題されるクイズアプリとなっています。
 Levelが1〜３まで用意されており、各5問ずつ出題されます。
 問題に対し4択で答えを選んでいただくと、答えと解説が出る仕様になっています。
